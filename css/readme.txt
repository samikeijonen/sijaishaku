Place additional css in this directory. There is is a couple of example .css files in this folder. 
It means that you can use those .css files when editing post/page and it overwrites your current style.css for that post/page. 
Look for "Stylesheet" metabox, when editing post/page. From there you can choose any additional stylesheets.

Note! Custom stylesheets must have the 'Style Name: Value' header like this.

/**
 * Style Name: Example 1
 */
